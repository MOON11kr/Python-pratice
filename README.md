# 🔥 Python Strings 

A structured collection of **Python string problems** commonly asked in coding interviews.  
Organized by difficulty to help you build strong fundamentals and progress to advanced patterns.

---

## 🧠 Level 1: Core Basics (Be Fast ⚡)

- Reverse a string  
- Check if a string is a palindrome  
- Count vowels in a string  
- Count frequency of each character  
- Remove duplicate characters from a string  
- Remove duplicates while preserving order  
- Find the first non-repeating character  
- Check if two strings are anagrams  
- Find the length of a string without using `len()`  
- Convert lowercase string to uppercase without using built-in functions  

---

## ⚡ Level 2: Intermediate (Very Common in Interviews)

- Count words in a string  
- Find the most frequent character  
- Replace spaces with `%20` (URLify problem)  
- Check if a string contains only digits  
- Find all substrings of a string  
- Find the longest word in a sentence  
- Remove all special characters from a string  
- Check if two strings are rotations of each other  
- Find the second most frequent character  
- Capitalize the first letter of each word  

---

## 🚀 Level 3: Advanced / Tricky (Where Most People Struggle)

- Find the longest substring without repeating characters  
- Check if a string is a valid shuffle of two strings  
- Find all permutations of a string  
- Implement string compression (`aaabb → a3b2`)  
- Find the longest palindromic substring  
- Group anagrams from a list of strings  
- Check if a string follows a given pattern  
- Minimum window substring problem  
- Convert string to integer (like `atoi`)  
- Check if two strings differ by only one edit
